# 🎃 万圣节关心弹窗 Halloween Care Messages

一个充满温馨与惊喜的万圣节主题 Python 小程序，通过可爱的弹窗向你关心的人传递温暖！

## ✨ 功能特性

- 🎁 **粉色礼物弹窗**：程序启动时首先显示一个温馨的粉色主题礼物弹窗
- 🎃 **万圣节主题弹窗**：随后弹出 20 个随机位置的万圣节主题弹窗
- 🎨 **多彩配色方案**：6 种精心设计的万圣节配色组合（橙色、紫色、黑色等）
- 💬 **多样化关心话语**：包含 15+ 条温馨的关心信息
- 🎯 **随机位置**：每个弹窗随机出现在屏幕不同位置，增添趣味性
- 🖱️ **便捷交互**：点击文字或按钮即可关闭弹窗
- 🧵 **多线程支持**：使用多线程技术实现流畅的弹窗体验

## 📦 打包说明

本项目使用 PyInstaller 进行打包，配置文件为 `hello.spec`。

### 打包步骤

1. **安装 PyInstaller**
```bash
pip install pyinstaller
```

2. **执行打包**
```bash
pyinstaller hello.spec
```

3. **获取可执行文件**
打包完成后，可执行文件位于 `dist/hello.exe`

### PyInstaller 配置说明

- `console=False`：无控制台窗口模式运行
- `upx=True`：启用 UPX 压缩，减小文件体积
- 单文件模式打包，方便分发

## 📝 自定义配置

你可以根据需要自定义以下内容：

### 修改关心话语

编辑 `hello.py` 中的 `care_messages` 列表：

```python
care_messages = [
    '你的自定义消息 1',
    '你的自定义消息 2',
    # ... 添加更多
]
```

### 修改配色方案

编辑 `halloween_colors` 列表中的颜色代码：

```python
halloween_colors = [
    ('#背景色', '#文字色'),
    # ... 添加更多配色
]
```

### 调整弹窗数量

修改 `hello.py` 第 157 行的循环次数：

```python
for i in range(20):  # 修改这个数字
```

## ⚠️ 注意事项

- 程序会同时弹出多个窗口，建议在非工作时间使用
- 弹窗会随机出现在屏幕各个位置
- 点击任意弹窗的文字区域或关闭按钮即可关闭
- 如需紧急关闭所有窗口，可使用任务管理器
